# localcontext

Утилита для получения информации о локальном проекте: дерево файлов, сведения о системе и копирование исходного кода файлов проекта в один текстовый файл.

## Описание

Проект предназначен для:
- генерации структуры (дерева) проекта;
- вывода информации об операционной системе;
- сбора кода всех файлов проекта в один файл;
- нумерации строк при необходимости.

Все результаты сохраняются в указанный выходной файл.

## Установка и запуск

1. **Склонируйте репозиторий:**

   ```bash
   git clone https://github.com/redderi/localcontext.git
   cd localcontext
   ```

2. **Создайте виртуальное окружение (рекомендуется):**

   ```bash
   python -m venv venv
   source venv/bin/activate  # Linux/macOS
   venv\Scripts\activate     # Windows
   ```

3. **Запуск:**

   ```bash
   python main.py [опции]
   ```

## Аргументы командной строки

| Аргумент           | Описание                                    |
|--------------------|---------------------------------------------|
| `-t`, `--tree`     | Добавить дерево проекта в выходной файл     |
| `-s`, `--system`   | Добавить информацию о системе               |
| `-f`, `--filename` | Указать имя выходного файла                 |
| `-n`, `--number`   | Задаёт разрядность нумерации строк          |

**Примеры запуска:**

- Сохранить дерево проекта и сведения о системе:

  ```bash
  python main.py --tree --system
  ```

- Сохранить только код файлов с нумерацией строк:

  ```bash
  python main.py --number 4
  ```

- Указать имя выходного файла:

  ```bash
  python main.py --filename output.txt
  ```
